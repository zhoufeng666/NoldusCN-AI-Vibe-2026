DanioScope Basler Recorder
用于与 Noldus DanioScope 配合的 Basler 工业相机录制工具。

功能特性
Basler GigE 相机控制和录制
实时预览画面
MP4 格式视频录制 (MPEG-4 编码)
相机自动检测和诊断
可调节分辨率、帧率、比特率
Apple 风格 UI 设计
技术栈
C# / .NET Framework 4.7.2
Basler Pylon SDK 7.5
x64 Windows 10/11
依赖项
由于文件较大，以下依赖项需要手动下载：

Basler Pylon SDK (必需)

MPEG-4 Supplementary Package - 下载链接

前置要求
Visual Studio 2019/2022
.NET Framework 4.7.2 SDK
